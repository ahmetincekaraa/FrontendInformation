CSS (Cascading Style Sheets)

INLINE CSS

HTML tagları içinde style etiketi ile kazandırılan özelliklere denir. Çok fazla yer kapladığı için, kod karışıklığı ve tekrarı oluştuğu için çok kullanışlı değildir.

INTERNAL CSS

<head> etiketi içinde style ile özellik kazandırma yöntemidir. Class veya Id isimleriyle özellik verilebilir. Tag ismi ile özellik verirsek kullandığımız tüm taglar etkilenecektir.

EXTERNAL CSS

Ayrı bir CSS dosyası açıp orada tüm özellikleri verdikten sonra <head> etiketi içerisinde <link href=""> şeklinde dosyamızı tanımlayarak özellik kazandırma yöntemidir.

Id Etiketi; En etkili etikettir. Id sadece bir defa verilebilir. Style özelliğini '#' işareti ile verilir.

Class Etiketi; İkinci etkin etikettir. Style özelliğini '.' işareti ile verilir.

DISPLAY

Yan yana gelen bu nesnelere inline etiket, alt alta gelen etiketlere ise block etiketler denir.


Display için none | inline | inline-block | block elementleri mevcut.


  • display: block; alabileceği maksimum genişlik değerini alır ve tüm satırı kaplar. Kendisinden sonra gelen element bir sonraki satıra geçer.
  • display: inline; sadece kendisi kadar yer kaplar inline özelliğine sahip element Inline elementler aynı satırda sıralanırlar. Inline etiketlere genişlik ve yükseklik veremeyiz.
  • display: inline-block; inline-block özelliği block ile inline özelliklerinin bir birleşimi olarak ifade edilebilir. Yani, elementler aynı satırda konumlandırılırken yükseklik ve genişlik özelliklerine de müdahale edebilmekteyiz. Bu sebeple çoğu zaman layout sürecinde (menu, card, slider vb.) aktif olarak bu özellikten faydalanmaktayız.

Flexbox

  • flex-direction; Elemanların sıralanma yönlerini belirlemek için kullanılır.
    1. row: → Elemanları soldan sağa yatay sıralar. (Default)
    2. column: → Elemanları yukarıdan aşağı dikey sıralar.
  • flex-wrap Eklenen elemanlar kapsayıcı genişliğini aştığında yeni satırda başlaması için kullanılır.
    1. nowrap: → Elemanlar tek satırda soldan sağa sıralanır. (Default)
    2. wrap: → Elemanlar kapsayıcı genişliğini aştığında yeni satırda sıralanmaya başlar
  • Justify-content: → Öğelerin yatay eksende hizalanmasını ayarlamak için kullanılır.
  • Align-items: → Öğelerin dikey olarak hizalanmasında kullanılır.

JavaScript (JS)

JavaScript; web sayfalarında yaygın olarak kullanılan, geliştiricilerin sayfaları dinamik hale getirip tarayıcı ile kullanıcı arasındaki etkileşimi oluşturmak için kullandığı bir programlama dilidir.


JavaScript işlevleri, sosyal medya akışlarını yenilemekten animasyonlar ve etkileşimli haritalar göstermeye kadar, bir web sitesi kullanıcısının deneyimini iyileştirebilir.

JavaScript Array Metodları

JavaScript dizileriyle çalışırken kullanılan array metodları vardır. Bu metodlar; verileri işlemek, güncellemek, silmek gibi önemli işlevleri üstlenir. Kullanılan temel metodlar şunlardır;


  • push()
  • pop()
  • shift()
  • unshift()
  • find()
  • some()
  • every()
  • sort()
  • includes()
  • slice()
  • map()
  • filter()
  • reduce()
  • forEach()
  • indexOf()
  • lastIndexOf
  • reverse()
  • concat
  • join()
  • toString()

push()

Bir dizinin sonuna eleman veya elemanlar ekler ve dizinin yeni halini döndürür.

pop()

Bir dizinin son elemanını kaldırır ve kaldırılan elemanı döndürür.

shift()

Bir dizinin ilk elemanını kaldırır ve kaldırılan elemanı döndürür.

unshift()

Bir dizinin başına eleman veya elemanlar ekler ve dizinin yeni halini döndürür.

find()

Dizi içerisinde arama yapar. Belirlenen kriterleri sağlayan ilk değeri döndürür. Eğer kriteri sağlayan değeri bulunmaz ise 'undefined(tanımsız)' değerini döndürür.

map()

Dizi içerisindeki tüm elemanları bir işlemden geçirmek için kullanılır. Her bir eleman tek tek işlenir, geriye ne döndürüleceği belirlenebilir. Ve sonunda ise, işlemden geçen öğelerden oluşan bir dizi geriye döner.

filter()

Bir dizide belirlediğimiz koşulu sağlayan elemanları filtreler, istediğimiz koşula sahip olan elemanlarla yeni bir dizi oluşturulur.

forEach()

Belirlediğimiz bir işlevde döngü oluşturmamızı ve bu döngüyü sırayla çalıştırmamızı sağlar. Her elemanı bir defa döndürür.

indexOf()

Bir dizide belirlediğimiz elemanın bulunduğu ilk indeksi döndürür. Eleman dizide bulunmuyorsa '-1' değeri döner.

toString()

Kullanılan değişkenin değerini string olarak değiştirmek için kullanılır.

React

React; kullanıcı arayüzü oluşturmak için en çok tercih edilen JavaScript kütüphanesidir.Facebook tarafından geliştirilen bu kütüphane; Netflix, WhatsApp, Instagram gibi büyük şirketler tarafından da kullanılmaktır. Kullanımı kolay, açık kaynak kodlu bir kütüphanedir.

React Projesi Oluşturma

  1. npx create-react-app proje-ismi : Şeklinde projemizi oluşturuyoruz.
  2. cd proje-ismi : Şeklinde oluşturduğumuz projemizi seçiyoruz.
  3. npm start : Şeklinde projemizi ayağa kaldırır (çalıştırır).

Component (Bileşen)

React, component tabanlı bir yaklaşımı benimsediği için componentler çok önemlidir.

Component, React'ın en küçük yapı taşlarından biridir. Birbirinden bağımsız ve yeniden kullanılabilir kod parçalarıdır ve her componentin kendine özgü görevi vardır.

Functional Components

Function components; özünde bir JavaScript fonksiyonudur. Kullanılan en yaygın component oluşturma metodudur. İçinde bulunan return ifadesi ile geriye React elementi(JSX) döndürülür.

Class Components

Class Component ES6 class yapısını temel alır.React.Component Class’ından türetilir ve bu class içinde render fonksiyonu bulunur.Class component’leri, daha karmaşık durum yönetimi, yaşam döngüsü olayları ve daha fazla işlevsellik gerektiren durumlar için kullanılır.

Yaşam Döngüsü; Yazılım da aslında bir üründür ve ürünlerin bir yaşam süreci vardır. Yazılımın yaşam döngüsü tek yönlü veya doğrusal değildir ve birkaç temel aşamadan oluşur. Bu aşamalar şunlardır: Planlama, Analiz, Tasarım, Gerçekleştirme ve Bakım. Yani yazılım projelerinin analiz ve planlanmasından başlayarak kullanımı boyunca süren bir döngüdür diyebiliriz.

render; bileşenin HTML'sini sayfaya işlemek için kullanılır. Bir bileşenin durumu değiştiğinde, bileşen kendini yeniden işler, yani bileşenin HTML'sinin geçerli durumla güncel olduğundan emin olmak için render() yöntemi tekrar çağrılır.

ES5 / ES6; ES5, 2009'da yayınlanan bir JavaScript sürümünün kısaltmasıdır, ES6 ise 2015'te yayınlanan bir JavaScript sürümünün kısaltmasıdır.

Real DOM - Virtual DOM

Real DOM (Gerçek DOM); Tarayıcıda bulunan ve web sayfasının temsil edildiği gerçek HTML ağacıdır, ancak doğrudan değişiklik yapmak işlemci kaynaklarını tüketebilir ve performans sorunlarına yol açabilir.

Virtual DOM (Sanal DOM); React’in performansı artırmak için kullandığı bir kavramdır. Web uygulamasının kullanıcı arayüzünü temsil eden hafızadaki bir veri yapısıdır.
